# MAPID.io

1. Go to [https://iot.mapid.io/](https://iot.mapid.io/) and creata an account
2. In the `https://mapid.io/dashboard` click Create Device
3. Click `NEW DATASET` to create dataset
4. Click `API-KEY` to get the api-key
